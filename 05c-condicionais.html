Comandos condicionais

São estruturas de controle de fluxo que permite que o codigo tome decisões

Elas funcionam avaliando uma expressão lógica(condição) que resulta em verdadeiro (true)ou falso (false).

Com base nesse resultado, nosso script toma decisão e executa as ações conforme o resultado.

Exemplos de cenarios que envolvem lógica condicional

  • Processo login: dados certos? entra...errados? não entra!
  • Vitrine de produto: tem estoque? mostra o produto...senão, não mostra!
  • Cadastro de nha segura: o valor digitado atende os criterios? se sim, efetiva o cadastro....senão, não efetiva
